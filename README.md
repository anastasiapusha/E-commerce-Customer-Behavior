# E-commerce-Customer-Behavior
End-to-end data analysis project: cleaning and analyzing e-commerce customer data with SQL and building interactive dashboards in Power BI/Tableau to generate business insights.

## 🎯 Objectives
- Analyze customer spending patterns
- Identify high-value customers
- Evaluate the impact of discounts
- Understand customer retention behavior

## 📁 Dataset
The dataset includes:
- Customer demographics (age, gender, city)
- Purchase behavior (total spend, items purchased)
- Engagement metrics (rating, satisfaction)
- Recency (days since last purchase)

## 🛠 Tools Used
SQL (data cleaning & analysis)
Power BI / Tableau (dashboard)
Excel / Python (optional preprocessing)
